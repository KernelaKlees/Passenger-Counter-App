# Passenger Counter App

A simple JavaScript project that counts how many people enter a location.  
You can increment the counter and save each entry.  
This mini‑project was originally inspired by an old FreeCodeCamp JavaScript exercise.

## 🧠 Features
- Increment the number of people entering  
- Save the current count  
- Reset automatically after saving  
- Display previous entries  

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript

## 📜 How It Works
- Click **Increment** to increase the counter  
- Click **Save** to store the current count  
- The counter resets to zero after saving  
- Saved entries appear in the “Previous entries” section
